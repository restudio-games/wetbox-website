About Wetbox

Wetbox Engine is a game development framework based entirely on VS Code and aimed at serious game developers. The engine has no built-in libraries; developers install and configure the libraries themselves according to their project needs.

Restudio Games' role is to prepare these libraries to make getting started easier, but they are not part of the engine itself.

This approach gives developers full control over their game's architecture, performance, and dependencies, allowing for highly optimized and tailored game development experiences without the overhead of unnecessary features.

Wetbox is designed for developers who want to understand every aspect of their game's technical implementation and prefer to work with raw, unabstracted libraries and APIs.
